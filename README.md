# Manuka — Bienestar y Salud Preventiva

Sitio web de una página para **Manuka** (@manukard), centro de bienestar y salud preventiva en Santo Domingo, República Dominicana.

**Demo:** https://daniel-projectt.github.io/manuka-wellness/

## Stack

HTML + CSS + JS puro (sin dependencias). Tipografía: Fraunces + Figtree (Google Fonts).

## Estructura

- `index.html` — todo el contenido (una página, secciones ancladas)
- `css/style.css` — identidad visual (paleta miel/cacao, hero con resplandor que respira, panal sutil)
- `js/main.js` — nav sticky, menú móvil, revelado al hacer scroll
